# Employee Salary Dashboard

## Overview

This project is an interactive HR Analytics dashboard developed using Power BI. It provides insights into employee salary distribution, department performance, workforce demographics, and experience levels to support HR decision-making.

## Objective

The objective of this dashboard is to help HR teams monitor employee information, identify salary trends, compare department performance, and analyze workforce distribution.

## Tools Used

- Power BI
- Microsoft Excel
- DAX
- Data Modeling

## Key Metrics

- Total Employees
- Average Salary
- Highest Salary
- Total Payroll
- Average Experience

## Dashboard Features

- Department-wise employee analysis
- Gender distribution
- Salary distribution by department
- Experience analysis
- Employee details table
- Interactive slicers and filters

## Business Insights

- Compare salary distribution across departments.
- Identify departments with the highest payroll.
- Analyze employee experience levels.
- Monitor workforce distribution using interactive visualizations.
